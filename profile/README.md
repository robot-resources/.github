# Robot Resources                                         

> Tools for AI agents. Humans have HR. Agents have RR.

We build tools for AI agents and any software that makes LLM API calls — chatbots, RAG pipelines, AI-powered apps, internal tools.

### Router — Intelligent model routing

Routes each prompt to the cheapest model that can handle it. 14 models across OpenAI, Anthropic, and Google. **60-90% cost savings** with zero quality loss. Runs as a local proxy — your API keys never leave your machine.

### Scraper — Token compression for web content

Fetches any URL, strips noise, returns clean markdown with token count. **70-80% fewer tokens** per page. No API keys, no config.

### Install

npx robot-resources

One command. GitHub login. Router runs as an always-on service. MCP auto-configured. Done.
